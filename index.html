Do You have time to talk about Python?

I am an enthusiastic Python programmer, since it is easy to write, easy to read easy to use even for bigger software projects. As a computer vision researcher, I use a lot of numpy, scipy and matplotlib. Together with different deep learning frameworks they are really powerfull tools different computer vision tasks. Nevertheless, I also familiar with the Django web framework. You can find some of my open source code below.

Chainer Addons
